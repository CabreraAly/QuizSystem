# QuizSystem O.O.P. Project

## Description
A console-based Java quiz system that allows users to:
- Create questions
- Edit questions
- Delete questions
- View created quizzes
- Start and take the quiz with feedback

## Features
1. Add multiple-choice questions (a/b/c/d)
2. Edit and delete questions
3. Track correct and wrong answers during the quiz
4. Clear all questions
5. Works entirely in console

## How to Run
1. Compile the Java file:
   ```bash
   javac QuizSystem.java
